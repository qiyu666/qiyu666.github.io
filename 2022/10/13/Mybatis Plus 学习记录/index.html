Mybatis Plus 基础学习

本文最后更新于：2022年10月26日 下午

官网

国人基于 Mybatis 开发的开源框架

在 MyBatis 的基础上只做增强不做改变，为简化开发、提高效率而生。

整合 Spring Boot 使用

  • 创建项目

  • 导入依赖

  • 配置数据源和日志

  • 在项目启动类上添加 @MapperScan("") 注解

    • 指定 mapper 文件夹的位置，否则会报错 🔴

  • 创建数据库、表

    • 创建实体类

    • 创建 Mapper

      • 继承 MP 提供的通用 Mapper -> BaseMapper
      • 在 <?> 中指定 Mapper 所对应的实体类类型

  • 测试

    在相应的 Mapper 上右键 -> Go To -> Test -> Create New Test... -> ok

    这样就会生成对应的测试类进行测试

@TableFiled

映射数据表非主键字段名

用法

  • 添加在实体类里的除了 id 之外的属性名上

    • select：表示是否要查询该字段，如果值设置为 true 则查询，否则为 false 不查询，默认为 true

@Version

CRUD 操作

Insert

参数说明

类型 参数名 描述
T entity 实体对象

Delete

参数说明

类型 参数名 描述
Wrapper <T> wrapper 实体对象封装操作类（可以为 null）

参数说明

Map<String, Object> columnMap 表字段 map 对象
IPage <T> page 分页查询条件（可以为 RowBounds.DEFAULT）
